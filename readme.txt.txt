1.Install OS on Raspeberry pi B3+

2.Start up raspberry pi and connect a direct display on HDMI Port.

3.Install Python and add to path.

4.Copy python files from project Code to the respberry storage.

5.Install a python editor like spider on Raspberry.

6.Open TCP/IP files in the Editor and run.

7.Tcp client will be established on PI.

8.Run the attached camera.

9.Open the Arduino IDE.

10.Connect arduino to PC by USB cable.

11.Burn The arduino code on the arduino memory.

12.Connect bluetooth module with arduino and disconnect from pc.

13.Connect servo motor .

14.Attach arduino module to the door lock.

15.Run pycharm on PC.

16.Open TCP server file.

17.Connect system to WLAN.

18.Start the server.

19.Open live monitoring Window to view live cam.
